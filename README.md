<h1>Group Policy Management</h1>
In this tutorial, we observe Group Policy Management in Server Manager and explore creating folders and assigning permissions for desginated users or groups. <br />

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Server Manager
- Group Policy Management

<h2>Operating Systems Used </h2>

- Windows Server 2022

<h2>High-Level Steps</h2>

- Step 1: Login remotely to Windows Server
- Step 2: Open Group Policy Management in Server Manager
- Step 3: Create Folders to Share
- Step 4: Enable Group Policies
